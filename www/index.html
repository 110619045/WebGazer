WebGazer.js

Real Time Eye Tracking

WebGazer.js is an easy to use library which provides continuous gaze predictions as screen coordinates.

No Special Hardware

Other eye tracking programs often require specialized or expensive hardware components. WebGazer.js taps into normal, commodity grade webcams like you would find on a laptop and runs in the web browser.

Predictions From Only A Few Clicks

Once a user has clicked the screen a few times, WebgGazer.js will start giving eye predictions.

Creating from Source

Clone the repository and then cd into build, then run the build_library command.

git clone https://github.com/brownhci/WebGazer.git
cd build
./build_library
