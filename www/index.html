WebGazer.js

Democratizing Webcam Eye Tracking on the Browser

WebGazer.js is an eye tracking library that uses common webcams to infer the eye-gaze locations of web visitors on a page in real time. The eye tracking model it contains self-calibrates by watching web visitors interact with the web page and trains a mapping between the features of the eye and positions on the screen. It is written entirely in JavaScript and with only a few lines of code can be integrated in any website that wishes to better understand their visitors and transform their experience.

Real Time Eye Tracking

WebGazer.js is an easy to use library which provides continuous gaze predictions as screen coordinates.

No Special Hardware

Other eye tracking programs often require specialized or expensive hardware components. WebGazer.js taps into normal, commodity grade webcams like you would find on a laptop and runs in the web browser.
